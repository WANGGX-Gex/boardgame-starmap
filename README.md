# 🌌 BoardGame StarMap 桌游星图

一个交互式的 3D 桌游关系网络星图，将桌游、设计师、美术和出版商以星球的形式呈现在宇宙空间中，并用连线展示它们之间的关系。

## ✨ 在线体验

👉 **[点击这里打开星图](https://你的用户名.github.io/boardgame-starmap/)**

> 部署后请将上方链接中的 `你的用户名` 替换为你的 GitHub 用户名。

## 📖 关于本项目

本星图收录了 [BoardGameGeek](https://boardgamegeek.com)（BGG）综合排名前 1000 的桌游，以及与它们存在扩展、重制、兼容、包含等关系的关联桌游。这不是 BGG 上所有桌游的收录，而是聚焦于最核心的 Top 1000 及其关系网络。

星图中包含三种节点类型：

- 🔵 **桌游**（球形）—— 核心节点，大小与评分人数相关
- 🟡 **人物**（菱形）—— 设计师与美术
- 🟢 **出版商**（方形）—— 原始出版商与联合出版商

四种桌游间的关系连线：

- **扩展** —— A 是 B 的扩展包
- **重制** —— A 基于 B 进行了重制 / 改良
- **兼容** —— 两款游戏彼此独立，同时可以兼容游戏
- **包含** —— 常见于"大盒"，A 中包含了 B

## 🕹️ 功能

- **3D 星图漫游**：鼠标拖拽旋转、滚轮缩放、右键平移
- **搜索**：支持中英文搜索桌游、设计师、出版商，点击结果自动飞到目标节点
- **全局筛选**：按排名、重度、适配人数、出版地区筛选桌游
- **聚焦模式**：点击任意节点，查看与其直接相关的所有桌游、人物和出版商
- **详情面板**：展示评分、重度、玩家人数、设计师、美术、出版商，以及关联桌游
- **节点与连线控制**：可开关人物/出版商显示，切换出版商显示模式
- **响应式设计**：支持桌面端和手机端浏览

## 🛠️ 技术栈

- [3d-force-graph](https://github.com/vasturiano/3d-force-graph) —— 3D 力导向图
- [Three.js](https://threejs.org/) —— 3D 渲染引擎
- 纯 HTML / CSS / JS 单文件实现，无需构建工具

## 🚀 本地运行

由于浏览器安全策略，直接双击 `index.html` 打开可能无法加载 JSON 数据。推荐使用本地服务器：

```bash
# 方法一：Python（推荐）
cd boardgame-starmap
python -m http.server 8000
# 然后打开 http://localhost:8000

# 方法二：Node.js
npx serve .

# 方法三：VS Code
# 安装 Live Server 插件，右键 index.html → Open with Live Server
```

如果你不想用命令行，页面也提供了一个手动选择 JSON 文件的按钮作为备用方案。
